# Sail SJ
A Scramjet example with batteries included.

## What's in here:
- Scramjet files
- Libcurl files
- Baremux files
- A service worker
- Minimal HTML template
- `index.js` that inlines all runtime assets (Scramjet, Libcurl, Baremux, and wasm) into one JavaScript file

Feel free to switch out the transport, SW, and Wisp at will.
